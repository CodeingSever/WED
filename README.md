# เว็บให้ความรู้คอมพิวเตอร์

เว็บสแตติกสำหรับให้ความรู้พื้นฐานเกี่ยวกับคอมพิวเตอร์ เหมาะสำหรับผู้เริ่มต้น ที่ต้องการเข้าใจภาพรวมตั้งแต่ฮาร์ดแวร์ ซอฟต์แวร์ อินเทอร์เน็ต ความปลอดภัย ไปจนถึงการเริ่มต้นเขียนโปรแกรม

## โครงสร้างไฟล์

- `index.html` – หน้าเว็บหลัก รวมทุกหัวข้อความรู้ในหน้าเดียวเลื่อนอ่านได้
- `styles.css` – ไฟล์สไตล์ ทำให้เว็บดูสวย ทันสมัย และรองรับมือถือ
- `script.js` – สคริปต์เล็กน้อยสำหรับปุ่มเลื่อนกลับไปด้านบน

## วิธีเปิดใช้งาน

1. ไปที่โฟลเดอร์ `WED`
2. ดับเบิลคลิกไฟล์ `index.html` เพื่อเปิดในเว็บเบราว์เซอร์ (เช่น Chrome, Edge)
3. เลื่อนอ่านหรือกดเมนูด้านบนเพื่อข้ามไปยังหัวข้อที่ต้องการ
